# Non-Screen Functions

## Background & Automated Functions

| Function | Description | Trigger |
|----------|-------------|---------|
| **Auto Send Email** | Send email notification when application status changes | Application status update |
| **Send Interview Completed Email** | Send email to recruiter when candidate completes interview | Interview completion |
| **Notification System** | Create and send in-app notifications for user actions | Various user actions |
| **Job Matching Algorithm** | Automatically match candidates with jobs based on skills, experience, location | On-demand / Scheduled |
| **Candidate Matching for Jobs** | Find matching candidates for a job posting (VIP feature) | On-demand (Recruiter) |
| **Interview AI Analysis** | Analyze interview transcript using Gemini AI and generate feedback | Interview completion |
| **Interview Rule-Based Evaluation** | Evaluate interview using rule-based system | Interview completion |
| **Job View Tracking** | Automatically track when candidate views job detail page | Job detail page view |
| **Job Click Tracking** | Automatically track when candidate clicks on job listing | Job listing click |
| **Candidate Profile View Tracking** | Track when recruiter views candidate profile | Profile view |
| **Interview Expired Cleanup** | Automatically cleanup expired interviews | Scheduled task |
| **Payment Processing** | Process membership payment via payment gateway | Membership purchase |
| **Payment Status Check** | Check payment status from payment gateway | On-demand |
| **File Upload (Resume/Image)** | Upload and store files to cloud storage (R2) | File upload action |
| **Clerk User Sync** | Automatically sync Clerk user to database candidate profile | Candidate login |
| **Membership Usage Tracking** | Track interview usage and membership limits | Interview creation |
| **Membership Validation** | Check if candidate can perform interview based on membership | Before interview start |
| **Application Status Change Notification** | Create notification when application status changes | Application update |
| **Interview Access Code Generation** | Generate unique access code for interview session | Interview creation |
| **Job Slug Generation** | Auto-generate URL-friendly slug from job title | Job creation |
| **Company Slug Generation** | Auto-generate URL-friendly slug from company name | Company creation |
| **Review Statistics Calculation** | Calculate average rating and review statistics | Review create/update/delete |
| **Interview Feedback Generation** | Generate per-question feedback and overall assessment | Interview analysis |
